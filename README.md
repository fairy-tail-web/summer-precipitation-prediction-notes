# Summer Precipitation Prediction Notes

This repository contains code, notes, and exploratory experiments related to summer precipitation prediction and hydroclimate analysis.

## Overview
My research focuses on summer precipitation prediction, large-scale climate drivers, and subseasonal-to-seasonal predictability.  
This repository is used to organize related preprocessing scripts, model experiments, and analysis workflows in Python.

## Research Interests
- summer precipitation prediction
- climate variability
- large-scale circulation and teleconnection drivers
- interpretable machine learning for climate applications
- hydroclimate risk analysis

## Tools and Libraries
- Python
- NumPy
- Pandas
- Xarray
